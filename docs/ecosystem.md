# Alfred ecosystem

The Android app in this repository is one part of the Alfred ecosystem:

- [alfred-android](https://github.com/AIByJohannes/alfred-android) — this Android client
- [alfred-cli](https://github.com/AIByJohannes/alfred-cli) — command-line client
- [alfred-desktop](https://github.com/AIByJohannes/alfred-desktop) — desktop client

Keep these links permanent when adding cross-platform documentation so users can
move between clients without relying on a transient issue or chat link.
